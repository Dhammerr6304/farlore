## ðŸ“¦ MVP Scope
The MVP will be stripped down to this essential loop:

* **Create:** A writer can start a story with a title, a first chapter, and two options for Chapter 2.
* **Vote:** Users can connect their wallet and vote for one of the two options.
* **Resolve:** After a set time, the vote ends, and the winning option is revealed as the "canon" path.
* **Mint:** The final, complete story (Chapter 1 + winning Chapter 2) can be minted as an NFT on the Base Sepolia testnet.
